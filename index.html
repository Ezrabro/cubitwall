CubitWall

English : CubitWall is a web application that allows you to create a custom bitcoin wallet. You can`t change first letter of wallet, but you can change the rest of the letters.

فارسی : با استفاده از این برنامه می توانید کیف پول بیتکوین دلخواه خود را بسازید. شما نمی توانید برای اولین حرف کیف پول تغییر دهید، اما برای حرف بعدی می توانید تغییر دهید.
